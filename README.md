# Django Coffee Guide

Небольшой пет-проект на Django: «Гид по кофейням города».  
Цель — показать структуру аккуратного Django-приложения с PostgreSQL, поиском, фильтрами и простой версткой без тяжёлых фронтенд-фреймворков.

## Стек

- Python 3.12
- Django 5.1
- PostgreSQL (через Docker Compose)
- Poetry для управления зависимостями
- CSS без фреймворков (кастомная верстка)
- Django templates

## Возможности

- Главная страница со списком кофеен:
  - карточки с фото, рейтингом, районом и тегами;
  - кликабельная карточка ведёт на детальную страницу кофейни;
  - быстрый поиск по названию, району, адресу и фичам (`q` в query string);
  - фильтры:
    - «Открыто сейчас» (пока заглушка по полю `is_open_now`);
    - «Рейтинг 4+»;
    - «Wi-Fi»;
    - фильтр по уровню цен (`$`, `$$`, `$$$`).
- Страница «Все кофейни» (`/cafes/`) со списком всех заведений.
- Детальная страница кофейни:
  - основное фото;
  - теглайн;
  - адрес, район, город;
  - уровень цен и рейтинг;
  - описание (несколько абзацев текста).
- Админка Django с удобным списком кофеен:
  - поиск по названию/slug/адресу/фичам;
  - фильтры по городу, району, цене, Wi-Fi, еде и статусу «Открыто сейчас».
